# Gestor de Tareas PHP

Un sistema simple y funcional para gestionar tareas personales, desarrollado con **PHP**, **MySQL** y **Font Awesome**.

## Funcionalidades
* **Registro e Inicio de Sesión:** Sistema de usuarios seguro con sesiones.
* **Gestión integral de Tareas:** 
  * **Crear:** Agregar tareas rápidamente.
  * **Leer:** Visualizar la lista de tareas pendientes.
  * **Actualizar:** Editar el nombre de las tareas existentes.
  * **Eliminar:** Borrar tareas con confirmación.
* **Diseño Responsivo:** Interfaz limpia con iconos profesionales.

## Tecnologías utilizadas
* **Backend:** PHP 8.x
* **Base de Datos:** MySQL (MariaDB)
* **Frontend:** HTML5, CSS3 y Font Awesome para la iconografía.
* **Servidor:** XAMPP / Apache.

## Estructura del Proyecto
* `index.php` - Panel principal y listado de tareas.
* `login.php` / `registro.php` - Gestión de acceso.
* `agregar_tarea.php` - Lógica para insertar datos.
* `editar_tarea.php` - Formulario y lógica de actualización.
* `eliminar_tarea.php` - Lógica para borrar registros.
* `includes/` - Fragmentos de código reutilizables (header/footer).
